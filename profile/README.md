## UFV: Information Technology Services 

This is the GitHub organization for UFV's ITS department. We develop and maintain the software that powers UFV's digital infrastructure that supports the wider UFV community.

### Need IT Help? 
For technical support, please reach out to the IT Service Desk: 

- **Request IT Support**: need general IT support? [Submit a support ticket to get help](https://itservicedesk.ufv.ca/TDClient/52/ITServicesPortal/Requests/ServiceDet?ID=581)
- **Report an Outage**: notice a system is broken? [Submit an Incident](https://itservicedesk.ufv.ca/TDClient/52/ITServicesPortal/Requests/ServiceDet?ID=457)
- **Email**: [itservicedesk@ufv.ca](mailto:itservicedesk@ufv.ca)
- **Phone**: [604-864-4610](tel:604-864-4610)

### About This Organization

We use this space to develop internal tools, web apps, and integrations that support univiersity operations. Most repositories are for internal development, but some repositories may be public and free-to-use by the wider university community.
